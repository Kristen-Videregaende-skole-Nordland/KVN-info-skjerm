# KVN-info-skjerm
Kode for front og back end info skjerm i hallen


# start API
### må ha node installert på pc-en

## gå til api mappen
`cd API`

## Installer package
`npm i`

## Start server
`node server.js`

## endepunkter api

### værmelding nesna
`http://localhost:3000/api/weather/nesna`

### Kalender
`http://localhost:3000/api/today`
